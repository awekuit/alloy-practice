# Alloy TIPS

- Visual Studio Code とその Alloy extension を使う場合、画面上に現れる Execute リンクボタンは保存済みのコードの Execute である事に注意する。つまり、画面に表示されている内容を Execute したいならファイルを Save してから Execute する
- 再帰的な構造を扱う場合、モデルの表示にツリーを使うとわかりやすい事が多い
